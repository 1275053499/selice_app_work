更新说明：

v6.4.2

  1. 支持分享到微信小程序
  2. Twitter SDK支持bitcode

v6.4

  1. 新增微信精简版(缩减官方SDK体积99%)
  2. 新增微博精简版(缩减官方SDK体积96%)
  3. 新增QQ精简版(缩减官方SDK体积93%)
  4. 修正部分问题，提供稳定性

v6.3

  1. 新增FacebookMessenger分享
  2. Facebook可获取用户头像url
  3. 增加细化的集成错误提示
  4. 增加更多Swift示例(Demo)
  5. 修复部分问题，提高稳定性

v6.2.2

  1. 增加IDFA获取功能，可以根据自己需求选择下载

v6.2.1

  1. 修复微信好友分享量无法统计的问题
  2. 补充由于微信secret不正确导致的微信登录失败报错

v6.2.0

  1. 新增第三方登录平台：DropBox、VKontakte
  2. 新增第三方分享平台： 有道云笔记、印象笔记、Google+、DropBox、VKontakte、Pocket
  3. 增加图片水印功能
  4. 全面支持ATS

v6.1.2

  1. 更新QQ SDK，已支持i386 CPU架构
  2. 对第三方平台头像的url做https处理
  3. 微信/支付宝支付可正常自行处理回调url
  4. 全面支持HTTPS(新浪官方暂无支持)，并限制HTTP图片链接的分享

v6.1.1

  1. 支持分享到钉钉
  2. 增加短信及邮件分享结构体
  3. 增加分享面板添加自定义平台名及icon的接口
  4. 部分问题修复

v6.1

  1. 全新分享面板，支持多种样式，可定制修改
  2. 支持修改平台显示顺序，详见「进阶文档」-「分享面板UI」
  3. 加强自定义平台示例及说明，详见「进阶文档」-「用户自定义平台」
  4. 控制台提示更简洁，加强错误说明
  5. 全面支持HTTPS，并限制HTTP图片链接的分享
  6. 修复部分问题，提高稳定性

v6.0.3

  1. 增加Kakao、Pinterest及Tumblr平台
  2. 对分享参数依据第三方平台策略进行了处理
  3. 分享本地和网络图片更具兼容性
  4. 增加第三方平台支持类型说明，详见API文档
  5. 支持HTTPS
  6. 修复部分问题，提高稳定性

v6.0.2

  1. 修复SMS和Email分享图片的问题
  2. 修正获得用户信息时返回授权问题
  3. 增加swift的demo示例
  4. 支持Cocoapods，详情见U-Share SDK集成文档

v6.0.1

  1. 优化分享面板
  2. 分享未安装客户端或客户端版本不支持时会在分享面板隐藏平台
  3. 增加授权和分享提示的国际化文本支持
  4. 修复获取用户信息重复SSO授权的问题
  5. 修复新浪微博授权直接返回App的问题
  6. 新增API可隐藏平台下指定类型，如微信收藏。

v6.0.0

  1. 全面重构社会化分享SDK，组织更清晰，第三方平台添加更便捷
  2. API接口相对5.x版本更加简洁易用，授权、分享和获取用户信息接口独立
  3. 核心库较老版本有更小的size
  4. 开源UI，分享面板可灵活定制
